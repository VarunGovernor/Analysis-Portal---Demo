# CyberTrace — Bank & CDR Intelligence Portal

**CyberTrace** is an advanced intelligence portal designed specifically for cybercrime investigation. It serves as a unified command center for law enforcement investigating officers to analyze bank transactions, trace money trails, and inspect Call Detail Records (CDR).

## Key Features
- **Bank & CDR Analysis**: Easily upload Bank Statement CSVs and Call Detail Records to run automated pattern detection.
- **Visual Intelligence**: Generates transaction distribution (Pie charts) and transaction volume history (Bar charts).
- **Money Trail Visualization**: Automatically traces end-to-end fund movement across multi-layered shell accounts.
- **Red Flag Alerts**: Systematically identifies suspicious patterns such as high midnight transactions, dormant account activations, and rapid round-trip transfers.
- **Case Management**: Organize active and archived cybercrime investigations, track assigned officers, and update statuses.
- **Report Generation**: Export comprehensive PDF and DOCX reports summarizing the analysis and money trail to assist legal proceedings.

## Layout & Structure
- **Dashboard**: The primary operational view containing upload utilities, transaction statistics, chart visualizations, money trail graph, anomaly red flag alerts, and a filterable transaction table.
- **Analysis Engine**: Deep-dive modules focusing on CDR Pattern Engines, entity relationship mapping (Network Graphs), and geolocation traces.
- **Cases Hub**: An overview of active/archived investigations with linked complaint types and tracking metadata.
- **Reports**: An archival view of all generated evidence documents.
- **Settings**: Portal configurations including UI preferences like Dark Mode and Compact View.
